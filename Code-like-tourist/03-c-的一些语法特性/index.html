以前用 char[] 的方式处理字符串很繁琐。
string 类，定义、拼接、输出、处理都很简单
string 只能用 cin 和 cout 处理，无法用 scanf 和 printf 处理：

  • cin读入字符串的时候，是以空格为分隔符的
    • 如果想读入一整行的字符串，就需要使用 getline
  • s 的长度可以用 s.length()获取
    • 字符数量，不包含\0
